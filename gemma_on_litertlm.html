About LiteRT LM 關於 LiteRT LM

LiteRT LM is Google's on-device LLM runtime, accessed via MediaPipe's LlmInference API. It handles tokenization, KV-cache, and autoregressive decoding internally. The raw LiteRT Web SDK (@litertjs/core) is a low-level tensor runtime that cannot run LLMs directly.
